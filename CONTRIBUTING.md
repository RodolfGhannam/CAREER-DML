# Contributing to CAREER-DML

Thank you for your interest in contributing to this research project.

## How to Contribute

This is an active PhD research project at Copenhagen Business School (CBS), Department of Strategy and Innovation. Contributions are welcome in the following areas:

1. **Bug Reports:** If you find a bug in the pipeline, please open an issue with a detailed description and steps to reproduce.

2. **Methodological Suggestions:** If you have suggestions for improving the causal inference methodology, embedding architecture, or validation framework, please open an issue for discussion.

3. **Data Integration:** If you have access to administrative labor market data (e.g., Danish register data, LEHD, or similar) and are interested in collaboration, please reach out.

## Code Style

- Follow PEP 8 conventions.
- Use type hints where possible.
- Document all functions with docstrings.
